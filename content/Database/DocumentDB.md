# Amazon DocumentDB

Amazon DocumentDB is a fully managed, scalable, highly available document database service that supports MongoDB workloads.

## Key Features
- **MongoDB Compatibility**: Drop-in replacement for MongoDB (uses the same drivers and tools).
- **Architecture**: Separates compute and storage. Storage automatically scales up to 128 TB.
- **Replication**: Supports up to 15 low-latency read replicas across multiple AZs.

> [!exam]
> Choose DocumentDB when a scenario requires migrating an existing **MongoDB** database to AWS without rewriting application queries, or when needing a fast, scalable **document database**.
