# AWS Compute Optimizer

AWS Compute Optimizer recommends optimal AWS resources for your workloads to reduce costs and improve performance by using machine learning to analyze historical utilization metrics.

## Key Features
- **Supported Resources**: EC2 instances, Auto Scaling groups, EBS volumes, and Lambda functions.
- **Recommendations**: Recommends scaling up (if bottlenecked) or scaling down (if over-provisioned).
- **Cross-Account**: Can aggregate recommendations across an organization using AWS Organizations.

> [!exam]
> Look for Compute Optimizer when a scenario asks how to **right-size** an existing fleet of EC2 instances or Lambda functions automatically based on historical usage patterns.
