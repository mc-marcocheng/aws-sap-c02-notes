# DMS Practice Questions

> [!question]
> Which AWS service would simplify the migration of a database to AWS?
> 1. AWS Storage Gateway
> 2. AWS Database Migration Service (AWS DMS)
> 3. Amazon Elastic Compute Cloud (Amazon EC2)
> 4. Amazon AppStream 2.0
> 
> > [!success]- Answer & Rationale
> > **Answer: 2**
> > **Rationale:** [[DMS Overview|AWS Database Migration Service (DMS)]] is specifically designed to simplify and automate the migration of databases to AWS. It supports both homogeneous and heterogeneous migrations, ensuring minimal downtime through continuous data replication ([[DMS Overview|CDC]]). [[Storage Gateway Overview|Storage Gateway]] is used for connecting on-premises environments to cloud storage. [[EC2 Overview|EC2]] provides resizable compute capacity but does not automate database migration. [[EC2 Overview|AppStream 2.0]] is a fully managed non-persistent application and desktop streaming service.
